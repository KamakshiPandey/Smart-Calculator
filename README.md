# 🧠 Smart Calculator

A modern web-based **Smart Calculator** with:
- Basic + Scientific Operations
- Unit Conversions
- Voice Input & Output
- Dark/Light Theme Toggle

## 🚀 Features
- ➕➖✖️➗ Basic arithmetic
- 📐 Square root, power, sin, cos
- 🔄 cm ↔ inch, kg ↔ lb conversions
- 🎤 Speak calculations & hear results
- 🌓 Dark/Light theme
